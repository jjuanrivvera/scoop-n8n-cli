# scoop-n8n-cli

Scoop bucket for [**n8nctl**](https://github.com/jjuanrivvera/n8n-cli) — a portable CLI for the [n8n](https://n8n.io) public API.

## Install (Windows)

```powershell
scoop bucket add n8n-cli https://github.com/jjuanrivvera/scoop-n8n-cli
scoop install n8n-cli
n8nctl init
```

The manifest in this bucket is updated automatically by
[GoReleaser](https://goreleaser.com) on each release of `n8nctl`.
